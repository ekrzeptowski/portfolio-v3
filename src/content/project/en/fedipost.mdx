## Libraries used:

- [Next.js](https://nextjs.org/) - React framework
- [Redux](https://redux.js.org/) - State management
- [RTK Query](https://redux-toolkit.js.org/rtk-query/overview) - An efficient data fetching and caching tool.
- [shadcn/ui](https://ui.shadcn.com/) - CSS framework
- [Zod](https://zod.dev/) - Validation

## Features

- Schedule posts to be published on Mastodon, Pleroma, Misskey, Firefish.
- Utilizes builtin scheduling API.
- No need to run a server apart from instance type detection (in order to avoid CORS issues).

## How to use

1. Go to [fedipost.vercel.app](https://fedipost.vercel.app/)
2. Click "Get started"
3. Type in your instance URL
4. Click "Submit"
5. Authorize the app
