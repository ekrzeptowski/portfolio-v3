## Libraries used:

- [NextAuth.js](https://next-auth.js.org/) - authentication for Next.js
- [Prisma 2](https://www.prisma.io/) - Next-generation [ORM](https://en.wikipedia.org/wiki/Object%E2%80%93relational_mapping) for Node.js and TypeScript

- [React query](https://react-query.tanstack.com/) - used for fetching data and state management
  ![React query data fetching](https://res.cloudinary.com/dn3ero0me/image/upload/v1618142939/blog/76TSHtdPQ5_zv8uln.gif)
- [React Hook Form](https://react-hook-form.com/) - used for URL input state management and validation
  ![Form validation](https://res.cloudinary.com/dn3ero0me/image/upload/v1618142465/blog/Ve8fTNwmD0_hechm1.gif)
- [Tailwind CSS](https://tailwindcss.com/) - A CSS framework used for styling
